# 90 days of Python

I decided to spend 90 days learning Python and this repository is where I'm keeping everything I work on along the way.

Some files are small exercises.  Some are experiments where I am trying to understand a new concept.  Rather than only showing polished projects, I want to keep a record of the whole learning process.

The repository is organised by topics, so I can easily look back and see what I have covered.  

## Why I'm doing this

I find the best way to learn is to show up consistently and practice.

The goal isn't to become an expert in 90 days.  The goal is to build a daily habit, keep learning, and getting a littl better each day.

By the end of this challenge I  hope to have a solid understanding of Python fundamentals, a collection of projects to look back on, and a much stronger foundation for whatever I learn next.
