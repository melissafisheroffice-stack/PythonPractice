# 90 days of Python

I decided to spend 90 days learning Python and this repository is where I'm keeping everything I work on along the way.

Some files are small exercises.  Some are experiments where I am trying to understand a new concept.  Rather than only showing polished projects, I want to keep a record of the whole learning process.

The repository is organised by topics, so I can easily look back and see what I have covered.  

## Why I'm doing this

I find the best way to learn is to show up consistently and practice.

The goal isn't to become an expert in 90 days.  The goal is to build a daily habit, keep learning, and getting a little better each day.

By the end of this challenge I  hope to have a solid understanding of Python fundamentals, a collection of projects to look back on, and a much stronger foundation for whatever I learn next.

## Where am I at?

* Days 1-12 (June 17th) I have completed a foundational course and covered variables, functions, arguments, conditionals, lists, loops, dictionaries and classes.
* Days 13+  I next plan to practice those topics further and start researching pytest (the python testing framework).

## UI Tests using Playwright

Install playwright in the terminal using the command: pip3 install playwright

Install the pytest plugin that has helpful fixtures and command line options: pip3 install pytest-playwright

Install the playwright browsers: python -m playwright install

## API Testing
